# Akademik-Plus
System zarządzania akademikiem wspierający administrację w przydzielaniu pokoi, rozliczaniu opłat oraz obsłudze zgłoszeń mieszkańców.
System ma umożliwiać prowadzenie pełnej ewidencji pokoi (liczba miejsc, wyposażenie, status dostępności) oraz przypisywanie studentów do konkretnych miejsc wraz z historią zamieszkania.

Należy uwzględnić mechanizm naliczania miesięcznych opłat za zakwaterowanie (z możliwością różnych stawek w zależności od standardu pokoju) oraz kontrolę płatności i ewentualnych zaległości.

Część studencka powinna umożliwiać zgłaszanie usterek technicznych oraz podgląd historii opłat i zamieszkania.
Panel administracyjny powinien zawierać zestawienia wolnych miejsc, raporty obłożenia oraz statystyki wykorzystania pokoi.

System powinien wspierać role użytkowników (student, administracja) oraz umożliwiać generowanie podstawowych raportów.
