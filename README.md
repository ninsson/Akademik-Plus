# Akademik-Plus
System do zarządzania akademikiem: przydzielanie pokoi, ewidencja miejsc (standard, wyposażenie) i historia lokatorów. Zawiera moduł opłat (różne stawki, kontrola zaległości) oraz panel zgłaszania usterek. Administracja otrzymuje raporty obłożenia i statystyki. Obsługa ról: student i administrator.
